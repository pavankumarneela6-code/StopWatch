# ⏱️ Stopwatch

A modern and responsive **Stopwatch Web Application** built using **HTML, CSS, and JavaScript**. It provides accurate time tracking with hours, minutes, seconds, and milliseconds, along with an attractive user interface.

---

## 🚀 Features

* ▶️ Start Stopwatch
* ⏸️ Stop Stopwatch
* 🔄 Reset Stopwatch
* ⏱️ Millisecond Support
* 💎 Modern Glassmorphism UI
* 🌈 Gradient Background
* ✨ Glowing Effects & Animations
* 📱 Responsive Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any web browser.
4. Click **Start** to begin the stopwatch.
